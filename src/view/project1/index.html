Step 1 Step 2 Step 3

Step 1

In this tutorial we are going to create a todo application with color categorization.
The first step is to render the todos into an unsorted list (ul).

We use a constructor to create unique todo objects. The created objects are assigned to array.

With the array map-function we can map the contents of the array to a string containing li's. This string is assigned to the innerHTML property of the ul.

See the Pen Tutorial #1 by etienne (@eGOB) on CodePen.

Step 2

To add a todo, we add two input elements to the markup. One text input, for the todo text and one color input for the color of the todo.

We also add a submit button. Add a eventlistener to this button, to listen to click events. As soon as the button is clicked, we push the new todo with the values entered by the user to the todos array. Then we call the earlier created render function, which re-renders the todo's.

See the Pen Tutorial #2 by etienne (@eGOB) on CodePen.

Step 3

In the last step we add the ability to remove a todo. For this the render function is extended and a new button is added to each li.

This button contains a call to a newly created remove function. To identify each todo we also need to extend the todo constructor. We add a new property called id this property get's filled with a unique id each time the constructor is called.

To remove a todo we now search the selected todo, with the find function on the array. With the found todo we can remove it by filtering the array with the filter function.

As always when the todo array is changed, we call the render function, to re-render the todos onto the page.

See the Pen Tutorial #3 by etienne (@eGOB) on CodePen.
